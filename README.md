# Sayı Tahmin Oyunu

Python ile geliştirilmiş, kullanıcıdan 1-100 arasında bir sayı tahmin etmesini isteyen interaktif oyun.

## Nasıl çalıştırılır

1. Bilgisayarınızda Python 3 yüklü olmalı.
2. Terminal veya Komut Satırı’ndan oyunun bulunduğu klasöre gidin.
3. Şu komutu çalıştırın:
   python oyun.py
